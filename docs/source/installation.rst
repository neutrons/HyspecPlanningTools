.. _installation:

+++++++++++++
Installation
+++++++++++++

* Create the development conda environment

   .. code-block:: bash

      $ conda env create

* Activate the environment

   .. code-block:: bash

      $ conda activate hyspecppt_dev

* Install the application in editable mode

   .. code-block:: bash

      $ pip install -e .

* Start the tool

   .. code-block:: bash

      $ hyspecppt
